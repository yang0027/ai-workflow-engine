### 数据流路径

```
用户操作 → 节点触发 → executeNode（纯分发）
                         ├── actionType='chat'    → ChatAdapter → /api/v1/llm/chat
                         ├── actionType='generate' → 图片 → /api/v1/image/generate
                         ├── actionType='fusion'   → 视频 → /api/v1/workflow/video/fusion
                         ├── actionType='tts'     → TTS → /api/v1/workflow/tts/clone
                         └── actionType='control'  → 自定义 → /api/v1/workflow/custom

※ 当前只有 PromptSourceNode 走 executeNode，其他节点直接 fetch
```
